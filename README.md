# 💰 Expense Tracker

Трекер расходов — GUI-приложение для отслеживания личных расходов с фильтрацией и статистикой.

---

## 📋 Описание

Приложение позволяет:
- ✅ Добавлять расходы с указанием суммы, категории, даты и описания
- ✅ Просматривать все расходы в виде таблицы
- ✅ Фильтровать расходы по категории и периоду
- ✅ Подсчитывать общую сумму расходов за выбранный период
- ✅ Экспортировать отчёт в текстовый файл
- ✅ Автоматически сохранять данные в JSON-файл

Все используемые модули входят в **стандартную библиотеку Python**:
